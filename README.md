# @briklab/bt 
## Binary Types for faster and small networking

> ## Info
> This will be merged to the organisation Kryft-Studios

**@briklab/bt** is a mini package used in briklab's networking; Used for a low-as-possible communication schematic. Used as a **low-latency, low-memory** communication structure.

# Installation

Usually, the format is:
```bash
[package manager] add|install @briklab/bt
```

## Not recommended for realtime stuff like chats; use @briklab/comcate


# Useful to know
This package only supports unsigned integers, signed integers and floating points to this date.
